# myhealth
1. 可以运行在我自己的iPhone上，我希望前端的界面简洁，但是需要耐看
2. 功能：读取我的健康app的数据，目前包括：resting energy和 active energy,以及睡眠的时间
3. 有一个我自行增加的选项是体重。另外可以上传早餐午餐晚餐的照片，并传送给ai的api获取三餐摄入能量，并且把能量显示出来。并且计算出今日的能量摄入，消耗，差值等等。
4. 每天根据这些数据发送给ai的api获取日报。
5. 每周根据这些数据获取周报。
6. 需要存储每日能量和体重能量的数据，并且类似iPhone健康app记录，并且以图（周月等等）显示。
7. 我的iPhone默认是英文
